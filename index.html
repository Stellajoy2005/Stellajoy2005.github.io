Assignments

1. Start a Website

In this assignment you will get started with building a website. You will create an HTML page, use an SFTP Client, and publish your file on a web server.

2. Citing Code

Follow a Tutorial and extend it with your own code. Use comments to indicate which lines are yours and which lines are from the tutorial. When you make larger projects you will combine various examples and tutorials together, and extend them to make your own work.

3. Text Adventure

Create a Webpage that includes P5.js and requires the visitors to input text to make it work. You can create a text game, animate a person's name, a grocery list application, a poetry generator, a text scrambler, a cypher generator, or anything else.
